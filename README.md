🎲 Tavla (Backgammon) JS Projesi
Staj dönemimde JavaScript mantığını ve DOM manipülasyonunu öğrenmek amacıyla geliştirdiğim interaktif bir Tavla oyunu projesidir.

🚀 Özellikler
Sürükle-Bırak (Drag & Drop): Taşları fare ile sürükleyerek hareket ettirebilirsiniz.

Zar Mekanizması: Rastgele zar atma ve çift gelme durumunda 4 hamle hakkı tanıma.

Sıra Takibi: Beyaz ve Siyah oyuncular arası otomatik sıra geçişi.

Kırık Taş Mantığı: Rakibin tekli pulunu "kırma" ve "bara" gönderme özelliği.

Dinamik Görsel: Taşların üst üste binme durumuna göre boyutlarının otomatik ayarlanması.

🛠 Kullanılan Teknolojiler
HTML5

CSS3 (Flexbox ve CSS Shapes kullanılarak tahta tasarımı yapıldı)

JavaScript (ES6+)

🎲 Backgammon JS Project (English)
An interactive Backgammon game developed during my internship to master JavaScript logic and DOM manipulation.

🚀 Features
Drag & Drop: Move your checkers smoothly across the board.

Dice Logic: Randomized dice rolls with "doubles" (4 moves) rule support.

Turn Management: Automatic switching between White and Black players.

Hit & Bar: Ability to "hit" an opponent's single checker and send it to the bar.

Responsive Scaling: Dynamic resizing of checkers when stacked on a point.

🛠 Tech Stack
HTML5

CSS3

JavaScript (ES6+)
